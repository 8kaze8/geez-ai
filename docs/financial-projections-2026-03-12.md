## 6. Unit Economics

### Customer Acquisition Cost (CAC)

| Yıl | Marketing Spend | New Users | CAC       |
| --- | --------------- | --------- | --------- |
| Y1  | $6,000          | 50K       | **$0.12** |
| Y2  | $24,000         | 150K      | **$0.16** |
| Y3  | $72,000         | 300K      | **$0.24** |
| Y4  | $180,000        | 500K      | **$0.36** |
| Y5  | $500,000        | 1M        | **$0.50** |

**Not:** CAC çok düşük çünkü organic growth (Product Hunt, ASO, viral gamification) dominant. Paid marketing büyüdükçe CAC artıyor ama hala çok düşük.

### Lifetime Value (LTV)

**Premium User LTV:**

```
Average monthly revenue: $5.50 (sub) + $2.56 (affiliate) = $8.06
Average retention: 14 months (Y3, %5 monthly churn)
LTV = $8.06 × 14 = $112.84

LTV:CAC ratio = $112.84 / $0.24 = 470:1 ✅
(Hedef: >3:1, Geez AI çok sağlıklı)
```

**Free User LTV:**

```
Average monthly affiliate revenue: $0.53
Average retention: 4 months
LTV = $0.53 × 4 = $2.12

Free user bile pozitif LTV ✅
```

### AI Cost per User per Month

| Tier    | Routes/mo | AI Cost | Maps Cost | Total |
| ------- | --------- | ------- | --------- | ----- |
| Free    | 3         | $0.87   | $0.045    | $0.92 |
| Premium | 8         | $2.32   | $0.12     | $2.44 |

Premium user geliri ($8.06) vs maliyeti ($2.44) = **%70 gross margin per user** ✅

## 11. Investment Thesis (Financial)

### Neden Bu Projeksiyonlar Gerçekçi

1. **Wanderlog benchmark:** 5 kişi, $2.4M/yr, 2.8M download → Geez AI Y3'te 500K aktif ile $1.6M gerçekçi
2. **Düşük başlangıç maliyeti:** API-based architecture, single developer, Supabase = minimum burn
3. **Affiliate dominant revenue:** Booking.com/Viator commission = proven model
4. **Gamification moat:** Retention artırır → LTV artırır → tüm projeksiyonu güçlendirir
5. **Konservatif varsayımlar:** %5 premium conversion (industry avg %3-5), %5 churn (industry avg %8)

### Valuation Indicators (Y5)

```
Revenue Multiple (SaaS): 8-12x ARR
$14.9M × 10x = ~$149M potential valuation

Revenue Multiple (Consumer App): 5-8x
$14.9M × 6x = ~$89M potential valuation

Conservative estimate: $50-100M by Y5
```
